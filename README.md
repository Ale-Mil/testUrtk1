#этот калькулятор нужен для решения простых операций сложение(+), вычитание(-), умножение(*), деление(/). 
     ▪ При выборе операции +: результат = число1 + число2
     ▪ При выборе операции -: результат = число1 - число2
     ▪ При выборе операции *: результат = число1 * число2
     ▪ При выборе операции /: результат = число1 / число2 
От пользователя требуется написание данных в переменные в виде цифр. Например 1 пользователь вводит числа 7 и 8, а второй вводит числа 5 и 2.

#Тест 2.1 при выборе операции -, калькулятор выведет -1.
#Тест 2.2 при выборе операции -, калькулятор выведет 3.

#Тест 3.1 при выборе операции *, калькулятор выведет 56.
#Тест 3.2 при выборе операции *, калькулятор выведет 10.

#Тест 4.1 при выборе операции /, калькулятор выведет 0,875.
